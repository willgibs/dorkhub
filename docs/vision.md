# dorkhub — Product Vision (board-approved 2026-07-22; amended 2026-07-23)

dorkhub is where developers **find tools they weren't looking for**. GitHub is
where you go when you know the name; dorkhub is the neighborhood you wander.

**Who it serves:** curators first (the silent majority — discover, save,
follow, organize), creators second (showcase, claim). Governance: Will = board
(approves vision, pricing, launch go, new spend, consent escalations); the
orchestrator agent = CEO (executes everything else to launch).

**Flywheel:** broad ingestion fills the gallery → curation traffic → creators
list and claim their pages. From P2.5, ingestion publishes directly behind an
automatic quality floor; moderation is retroactive.

**Principles:**
1. Discovery over lookup — search exists (⌘K utility) but is demoted; the
   product's weight is on browse/tags/serendipity/recommendation surfaces.
2. Stars import is the activation moment ("bring your stars"). GitHub Lists
   have no API (re-verified 2026-07-23) — excluded until that changes; no
   scraping. Users organize into dorkhub-native public collections (P3).
3. Quality is a FLOOR, not a throttle (amended 2026-07-23) — human review
   time must never gate volume. Every published card carries content
   (enrichment floor + og imagery); an AI screen, user reports, and
   retroactive moderation keep the bar. Reversible automation (AI may
   unpublish, logged); irreversible actions (delete + block) stay human.
4. Unclaimed honesty + consent — badge, claim flow, owner/repo blocklist
   honored by every ingestion path, delete-and-block on request, user-facing
   report mechanism (P2.5).
5. Cost discipline — $0/mo pre-launch (AI usage rides Vercel AI Gateway's
   recurring free credit); Vercel Pro + Spend Mgmt at launch; Supabase Pro
   only when forced (~1k READMEs); board notified pre-spend.

**Monetization:** featured slots in discovery surfaces only, clearly labeled.
Pricing = board gate before launch ships it.

**Launch bar:** pixel-perfect full product. Content bar ~150–300 quality
projects across major tags. "Slow monetizable growth over 15 minutes of fame."

**Road:** P1 gallery engine ✅ → P2 discovery mechanics + quality floor →
P2.5 self-running gallery (publish-all + AI moderation + reports) → P3 public
collections + rich pages + design-QA sweep → P4 featured slots + hardening +
dorkhub.com launch → P5 articles zone (post-launch v1.1: RSS-driven overview
gallery, link-out-first so authors keep their traffic, like/save parity, SEO
engine). Details: docs/plans/p*-*.md.
